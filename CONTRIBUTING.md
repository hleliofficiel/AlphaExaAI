## ğŸš€ Getting Started

### 1. Fork & Clone

Fork this repository first, then:

```bash
git clone https://github.com/hleliofficiel/AlphaExaAI.git
cd AlphaExaAI
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Create a Branch

```bash
git checkout -b feature/your-amazing-feature
```

### 4. Make Your Changes

Write clean, documented code. Follow our code standards (below).

### 5. Test

```bash
python -m pytest tests/
```

### 6. Commit & Push

```bash
git add .
git commit -m "feat: add your amazing feature"
git push origin feature/your-amazing-feature
```

### 7. Open a Pull Request

Go to [github.com/hleliofficiel/AlphaExaAI](https://github.com/hleliofficiel/AlphaExaAI) and open a PR with a clear description of your changes.

## ğŸ“œ Code Standards

| Rule | Details |
|------|---------|
| **Formatting** | Python: Black formatter (`black .`) |
| **Docstrings** | All public functions must have clear docstrings |
| **Type Hints** | Use Python type hints where possible |
| **Testing** | New features must include unit tests |

### Commit Message Format

```

feat: add new feature
fix: fix a bug
docs: update documentation
test: add tests
refactor: code refactoring
```
